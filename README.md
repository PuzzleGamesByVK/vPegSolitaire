# vPegSolitaire
vPegSolitaire covers many aspects and variations of Peg Solitaire! There are plenty of horizontal-vertical layouts, and hexagonal aswell. A big part of our Github / Vue3 / Firebase project is Lazlos Leap. So we have the intention to bring old classic marvels in the age of smartphones! A next plan is be to implement AI solver.
